# 🖥️ 워크스테이션 (Workstation) 설정 및 규칙

이 파일은 프로젝트 작업 환경의 일관성을 유지하기 위해 사용자와 AI가 공통으로 준수해야 할 규칙을 기록합니다.

## 📍 주 작업 경로 (Source of Truth)

- **공식 경로**: `c:\Users\namoo\Documents\라올실내건축_신규`
- **절대 금지 경로**: `g:\내 드라이브\...` (Google Drive)

## 🚨 [필수] 세션 시작 시 확인 사항 (AI 체크리스트)

1.  **경로 확인**: 터미널을 열면 즉시 `pwd`를 실행하여 현재 위치가 **C 드라이브 공식 경로**인지 확인하십시오.
2.  **G 드라이브 차단**: 어떤 상황에서도 `g:` 드라이브 하위의 파일을 참조하거나 수정하지 마십시오. (클라우드 동기화로 인한 파일 잠김 및 속도 저하 방지)
3.  **포트 고정**: 개발 서버는 항상 `3000`번 포트를 사용합니다. (`npm run dev`) 충돌 시 다른 포트로 이동하지 말고 기존 프로세스를 종료하십시오.

## 🛠️ 수동 설정 (사용자 가이드)

- VS Code에서 `File > Open Folder...`를 클릭할 때 반드시 `c:\Users\namoo\Documents\라올실내건축_신규`를 직접 선택하여 열어주세요.
- 만약 터미널이 G 드라이브에서 시작된다면, C 드라이브의 이 폴더를 다시 열어서 세션을 초기화해주십시오.
