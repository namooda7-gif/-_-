## 🚫 절대 금지 사항 (Forbidden Actions)

### 1. 구글 드라이브(Google Drive) 접근 금지
- **대상 경로**: `g:\내 드라이브\...` (또는 이와 유사한 모든 클라우드 동기화 경로)
- **이유**: 클라우드 동기화 중 파일 잠김, 속도 저하, 빌드 오류 및 경로 혼선을 유합합니다.
- **규칙**: AI는 어떤 상황에서도 `g:` 드라이브 하위의 파일을 수정하거나 참조해서는 안 됩니다.

### 2. 프로젝트 폴더 중복 생성 금지
- **대상 경로**: `c:\Users\namoo\Documents\laol-project\...`
- **규칙**: 모든 작업은 오직 `c:\Users\namoo\Documents\라올실내건축_신규`에서만 이루어져야 합니다. 하위 폴더에 새 프로젝트를 생성하거나 복사본을 만드는 행위를 엄격히 금지합니다.

## ✅ 필수 준수 사항 (Mandatory Rules)

### 1. 작업 시작 전 주소 확인
- 모든 세션 시작 시 `pwd` 명령어로 현재 위치가 `c:\Users\namoo\Documents\라올실내건축_신규` 임을 먼저 확인한 후 보고하십시오.

### 2. 포트 고정 (Standard Port 3000)
- 개발 서버는 반드시 `3000`번 포트를 사용합니다. 포트가 점유되어 있다면 다른 포트를 쓰지 말고 `taskkill /F /IM node.exe /T`로 정리 후 다시 `3000`번으로 띄우십시오.

### 3. 워크플로우 우선 참조
- 모든 작업 시 `.agent/workflows/project_setup.md`에 명시된 경로 정책을 최우선으로 따르십시오.
