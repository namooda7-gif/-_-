## 🚨 [절대 규칙] 프로젝트 단일화 관리

이 프로젝트의 성공적인 개발과 동기화를 위해 반드시 아래 규칙을 지켜야 합니다.

### 1. 주 작업 디렉토리 (The Only Source of Truth)
- **공식 경로**: `c:\Users\namoo\Documents\라올실내건축_신규`
- **금지 경로 1 (Google Drive)**: `g:\내 드라이브\...` 하위의 모든 폴더 접근 금지.
- **금지 경로 2 (Duplicated Folders)**: `c:\Users\namoo\Documents\laol-project\...` 하위 접근 금지.
- **금지 사항**: 이 폴더를 다른 하위 폴더 내부로 복사하거나 이동하지 마십시오. 폴더가 두 개가 되는 순간 AI와 사용자의 작업이 어긋나기 시작합니다.

### 2. VS Code 워크스페이스 설정
- **권장 작업**: VS Code에서 `File > Open Folder...`를 통해 **위의 공식 경로(`c:\Users\namoo\Documents\라올실내건축_신규`)를 직접** 여십시오.
- **장점**: 에디터 상단에 폴더 이름이 정확히 표시되어야 AI도 권한 오류 없이 터미널 명령을 수행할 수 있습니다.

### 3. AI 작업 시작 시 필수 체크리스트
매번 세션을 시작할 때 AI는 다음을 수행해야 합니다:
1.  `pwd` 명령어로 현재 경로가 공식 경로인지 확인.
2.  현재 작업 중인 파일들이 **절대로 `g:\내 드라이브` 하위가 아님**을 확인.
3.  공식 경로가 아닐 경우, 즉시 사용자에게 수정을 요청하고 작업을 중단.
4.  `.next` 캐시가 꼬였을 경우 `Remove-Item -Recurse -Force .next` 후 재시작.

### 4. 사이트 접속 문제 해결 (Standard Port)
- **Port**: 항상 `:3000`을 기본으로 사용합니다.
- **Troubleshoot**: 포트 충돌 시 `taskkill /F /IM node.exe /T`로 모든 프로세스를 정리한 후 다시 시작하십시오.

### 5. 폴더 관리 (Cleanup)
- 만약 작업 중 `laol-project` 등 중복된 폴더가 보인다면, 즉시 사용자에게 확인 후 삭제하여 혼선을 차단하십시오.
